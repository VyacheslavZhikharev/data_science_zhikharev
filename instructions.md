 # Инструкция по работе с проектом

Этот проект прогнозирует свойства композитных материалов и сравнивает несколько подходов: классические модели, нейросети и кластеризацию.

## Что находится в проекте

- `hw_data_composite/X_bp.xlsx` — основная часть признаков.
- `hw_data_composite/X_nup.xlsx` — дополнительные признаки по нашивке.
- `1_ВКР_Жихарев_Вячеслав_Сергеевич_Свойства_композитов_Разведочный_анализ_и_Предобработка.ipynb` — загрузка, объединение и очистка данных.
- `2_ВКР_Жихарев_Вячеслав_Сергеевич_Свойства_композитов_Модели.ipynb` — классические модели регрессии.
- `3_ВКР_Жихарев_Вячеслав_Сергеевич_Свойства_композитов_Нейросеть.ipynb` — нейросетевые модели.
- `4_ВКР_Жихарев_Вячеслав_Сергеевич_Свойства_композитов_Кластеризация.ipynb` — кластеризация и моделирование по кластерам.
- `app.py` — CLI-приложение для прогноза целевой переменной.

## Как устроен пайплайн (простыми словами)

1. Берем два Excel-файла из `hw_data_composite`.
2. Объединяем их по индексу (`INNER JOIN`) в одну таблицу `df`.
   После объединения автоматически сохраняем файл `Dataset_composites_inner.csv`.
3. Смотрим распределения, корреляции и качество данных.
4. Убираем явные ошибки (например, строку с нулями в ключевых признаках).
5. Нормализуем данные для части экспериментов.
6. Обучаем модели:
   - классические регрессоры;
   - нейросети;
   - модели внутри кластеров.
7. Сравниваем метрики (MSE, RMSE, MAE, R2) и делаем выводы.

## В каком порядке запускать ноутбуки

Рекомендуемый порядок:

1. `1_..._Разведочный_анализ_и_Предобработка.ipynb`
2. `2_..._Модели.ipynb`
3. `3_..._Нейросеть.ipynb`
4. `4_..._Кластеризация.ipynb`

## Запуск CLI-приложения

- Базовый запуск: `python3 app.py --target modulus`
- Прогноз прочности: `python3 app.py --target strength`
- Рекомендация соотношения: `python3 app.py --target ratio`
- С пользовательскими входными данными: `python3 app.py --target strength --input-json input.json`

Если `input.json` не передан, приложение использует медианные значения признаков.
